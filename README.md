Duke's Table - A Zomato Clone

Duke’s Table is a modern, high-performance Zomato clone built using Vite + React.js, styled with Bootstrap and Tailwind CSS for a sleek and responsive user experience.

Features:
Fast and optimized performance with Vite
Dynamic UI powered by React.js
Beautiful and responsive design with Bootstrap & Tailwind CSS
Restaurant listings, user reviews, and search functionality
🚀 Feel free to contribute and enhance Duke’s Table!
